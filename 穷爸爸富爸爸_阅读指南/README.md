# 穷爸爸富爸爸 · 阅读指南

罗伯特·T·清崎《穷爸爸富爸爸》个人阅读笔记，手机端优化。

## 打开方式

双击 **`index.html`**

## 目录结构

```
穷爸爸富爸爸_阅读指南/
├── index.html          ← 主阅读页面
├── _source/
│   └── 穷爸爸.txt      ← 原书文本
├── scripts/
│   └── build.mjs       ← 重新生成
└── README.md
```

## 章节结构（共12章）

1. 序言
2. 引言：富爸爸，穷爸爸
3. 第一课：富人不为钱工作
4. 第二课：为什么要教授财务知识
5. 第三课：关注自己的事业
6. 第四课：税收的历史和公司的力量
7. 第五课：富人的投资
8. 第六课：不要为金钱而工作
9. 克服阻碍（5大阻碍）
10. 开始行动（10条指南）
11. 还需要更多东西吗
12. 后记

每章末尾含：**经典金句 · 大师总结 · 行动要点**


| Tab | 说明 |
|-----|------|
| **原文** | 12个章节（序言+引言+6课+3实践篇+后记），每章末附大师总结 |
| **笔记** | 重点、难点、逻辑流 |
| **框架** | 两个爸爸对比 + 财商六课 |
| **读法** | 四遍阅读法 |
