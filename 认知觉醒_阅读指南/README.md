# 认知觉醒 · 阅读指南

周岭《认知觉醒》个人阅读笔记，手机端优化，随时阅读。

## 打开方式

双击 **`index.html`**，或在手机浏览器中打开此文件夹。

## 目录结构

```
认知觉醒_阅读指南/
├── index.html          ← 主阅读页面（双击打开）
├── images/             ← 书中插图
├── _source/            ← 原始资源
│   ├── 认知觉醒.epub   ← 原书文件
│   └── extracted/      ← EPUB 解压内容
├── scripts/
│   └── build.mjs       ← 重新生成 index.html
└── README.md
```

## 四个 Tab

| Tab | 说明 |
|-----|------|
| **原文** | 全书章节，手机点右上角「目录」跳转 |
| **笔记** | 重点、难点、逻辑流（竖向时间线） |
| **框架** | 全书结构图 |
| **读法** | 四遍阅读法 + 阅读要诀 |

## 重新生成

```bash
node scripts/build.mjs
```
